# FaceCensor Pro - Installationsanleitung
# ==========================================

# ─── Python-Pakete (pip) ───
# pip3 install PyQt5
# pip3 install numpy
# opencv-python ist auf Jetson Nano i.d.R. vorinstalliert
# Falls nicht: pip3 install opencv-python

# ─── System-Pakete (apt) ───
# sudo apt-get update
# sudo apt-get install -y python3-pyqt5
# sudo apt-get install -y python3-numpy
# sudo apt-get install -y python3-opencv

# ─── DNN Face Detector Modell ───
# Das Modell wird beim ersten Start automatisch heruntergeladen.
# Alternativ manuell in den Ordner "models/" legen:
#   - models/face_detector.caffemodel
#   - models/deploy.prototxt
# 
# Download-Links:
# caffemodel: https://raw.githubusercontent.com/opencv/opencv_3rdparty/dnn_samples_face_detector_20170830/res10_300x300_ssd_iter_140000.caffemodel
# prototxt:   https://raw.githubusercontent.com/opencv/opencv/master/samples/dnn/face_detector/deploy.prototxt

# ─── Starten ───
# python3 main.py

# ─── Tastenkürzel ───
# ESC    → Beenden
# S      → Screenshot
# R      → Aufnahme starten/stoppen
# SPACE  → Detection ein/ausschalten
